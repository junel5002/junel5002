  <!-- Core Project -->
  <h2>Core Project</h2>

  <h3>KeySense AI</h3>
  <p>
    KeySense AI is an <b>AI-powered music key detection web application</b> that analyzes
    audio signals in real time to accurately determine musical keys. The system applies
    signal processing techniques alongside machine learning models to extract pitch
    distributions and harmonic structures from raw audio.
  </p>

  <p>
    The application is engineered as a <b>full-stack AI system</b>, combining optimized
    backend inference pipelines with an interactive frontend for real-time feedback and
    visualization.
  </p>

  <p>
    <b>Role:</b> Sole Developer — model design, signal processing, backend APIs, frontend integration.
  </p>
